Safari Technology Preview 162 で CSS Nesting が来た！

ダウンロードしたら Safari Technology Preview 163 だった。(一つバージョンが上がっていた）
以下の HTML で動作確認ができた！🎉

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Nest CSS</title>
    <style>
        .wrapper {
            background-color: #eef;
            & p {
                color: #f00;
            }
        }
    </style>
</head>
<body>
    <div class="wrapper">
        <p>Example text</p>
    </div>
</body>
</html>
```
