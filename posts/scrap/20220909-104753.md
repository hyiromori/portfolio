Critical CSS に対する、現在の一般に広く普及した考え方に対する疑問をまとめた記事かな。

- CSS が本当にネックになっているのか確認する
- Critical ではない CSS は `<head>` に入れない
  - ただし `<body>` にある CSS は DOMContentLoaded イベント前後に適用されます
- 普通は Critical CSS を後から導入する必要はない
