2021年 防災の日

災害発生時の対応を家族で話す

非常時の行動

  1. 安全な場所に避難する
  2. 危険がなければ『自宅』で待機・生活する
  3. 『自宅』が危険な場合䛿『本村公園』に移動する
  4. 火災が広がっている場合は『免許センター』または『こども自然公園』に移動する
  5. 避難が長期になりそうな場合は『二俣川小学校』に移動する
  6. 『Gmail』『メッセージ』『LINE』などで安否確認を行う
  7. 『災害用伝言ダイヤル(171)』で安否メッセージを登録する

自宅近辺の緊急給水栓

  1. 二俣川小学校
  2. 万騎が原中学校
  3. 本宿中学校
  4. さちが丘小学校
  5. こども自然公園
